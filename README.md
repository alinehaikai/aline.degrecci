# Provador Virtual — GitHub Pages

## Publicar
1. Crie um repositório no GitHub.
2. Envie `index.html` e o arquivo `.nojekyll` para a raiz.
3. Vá em Settings > Pages.
4. Em Source, escolha `Deploy from a branch`.
5. Selecione `main` e `/(root)` e salve.

## Atenção às imagens
O `index.html` ainda referencia seis URLs temporárias do Figma. Para uma versão permanente, exporte as imagens e substitua as URLs no objeto `IMG`, próximo ao final do arquivo.
